OpenChain KWG

Linux Foundation OpenChain Project의 Subgroup인 OpenChain KWG(Korea Work Group)은
OpenChain KWG(Korea Work Group)은 Open Source 정신인 협업과 공유를 통해 모두가 효과적으로
Open Source Compliance를 달성하기 위한 방법을 고민하고 공유하기 위한 모임입니다!

Software를 개발하여 배포하는 한국 기업에서 Open Source Compliance 업무를 담당하는 분이라면 참여가 가능합니다. Mailing List에 가입하세요!
